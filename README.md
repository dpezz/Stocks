# Advanced Stock Analyzer

This project is a Streamlit app that provides advanced stock analysis with price predictions, news sentiment analysis, and volume analysis.

## Setup
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file in the project root and add your News API key:
   ```
   NEWS_API_KEY=your_api_key_here
   ```
4. Run the app: `streamlit run app.py`
